Software, libraries and Tools
Update: Bitty Data Logger 3.0
Version 3.0 adds an exciting and hopefully, very useful major new feature. It's now possible to capture, chart and export data from devices like sensors which you have connected to the micro:bits pins on its edge connector.
micro:bit equalizer (video demo)

microbit beacon finder
The micro:bit finds various of BLE Beacons, and display the ID to LEDs. Includes: iBeacon, AltBeacon, Eddystone-UID, Eddystone-URL.
